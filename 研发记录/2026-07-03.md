## 问题与风险

1. 如果过早依赖重型机器人训练框架，会被带入模型训练和硬件支持负担。
2. 如果完全自研机器人训练全栈，会超出当前资源能力。
3. 适配器边界必须清晰，避免本项目变成外部框架的包装层。
4. 完整专利-代码证据链不应进入 GitHub 开源仓库，否则可能暴露权利要求级保护点和审查应对策略。
