# worldmodels

真实世界经验闭环训练框架。

本项目面向机器人、Agent 和物理 AI 在真实场景中的部署后学习，提供空间语义、经验回写、概念归纳、任务翻译、执行恢复、人类偏好和技能包沉淀的基础框架。

第一期开源聚焦 P010-P015 六个经验闭环：

1. P010 空间语义自动生成。
2. P011 物理经验内化机制。
3. P012 概念归纳跨域迁移。
4. P013 任务语义翻译。
5. P014 执行恢复。
6. P015 人类偏好。

## 定位

worldmodels 不做机器人本体，不训练通用世界大模型，也不作为完整企业安全网关。

本项目关注的是：

> 机器人如何在真实任务中沉淀可复用经验、补救能力、偏好记录和技能包。

运行时安全治理可通过 Sayfos SDK 等外部治理层适配；worldmodels 第一期开源只提供必要引用字段和适配接口。

## 第一期开源内容

```text

demo_runtime/
```

其中 `schemas/` 提供第一期核心对象格式，`demo_runtime/` 提供最小运行时闭环。
