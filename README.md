# worldmodels

真实世界经验引擎，用于帮助机器人、自动化执行体和物理 AI 在真实场景中沉淀、复用和验证可执行经验。

本项目第一期开源聚焦 P010-P016 的公开产品层对象模型和最小 demo。它不训练通用世界大模型，也不绑定某一种机器人本体；它提供的是让机器人在真实环境中更快形成可复用技能的经验底座。

## 定位

worldmodels 关注的问题是：

> 机器人如何在真实任务中，把空间语义、执行过程、因果事实、失败恢复、人类偏好和技能包沉淀为可复用的经验资产。

当前公开层包含两类能力：

1. P010-P015：空间语义、经验记录、概念归纳、任务计划、执行恢复、人类偏好与技能包沉淀。
2. P016：过程模板、偶然绑定数据、连续状态变量、阶段跃迁、目标因果事实验证、最终成立状态回流。

其中 P016 是面向物理动作执行的底座层：过程模板定义可复用动作结构，偶然绑定数据将模板适配到当前环境，独立观测通道验证目标因果事实，最终成立状态再回流到阶段推进、恢复执行或后续动作触发。

## 快速验证

当前仓库提供两个最小 demo。

运行治理 demo：

```powershell
python .\demo_runtime\run_demo.py
```

预期输出摘要：

```json
{
  "total_steps": 6,
  "allowed": 2,
  "limited_allowed": 2,
  "blocked": 1,
  "metadata_only": 1
}
```

运行技能共创 demo：

```powershell
python .\demo_runtime\run_skill_cocreation.py
```

预期输出摘要：

```json
{
  "total_steps": 6,
  "experiences": 6,
  "recoveries": 1,
  "preferences": 2,
  "skill_steps": 5
}
```

## 开源内容

```text
schemas/
demo_runtime/
```

`schemas/` 提供第一期核心对象格式，`demo_runtime/` 提供最小运行时闭环。运行时安全治理可通过 Sayfos SDK 等外部治理层适配，本仓库第一期只保留必要引用字段和接口位置。

## 商业边界

第一期开源用于验证方法、建立对象格式和吸引共创。以下内容不属于第一期开源范围：

1. 完整空间治理实现。
2. 行业真实经验库。
3. 高价值技能包。
4. 企业控制台。
5. 商业治理策略引擎。
6. 私有化部署方案。
