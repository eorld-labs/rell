筛选口径：

本轮不是按“机器人、技能、迁移”等宽词直接判近似，而是按 P017 当前母案主链筛选：

```text
经验记录 / 经验不变量契约
+ 当前空间语义数据
+ 本体能力画像 / 跨本体适配判断
+ 任务期 RuntimeWorldState
+ 绑定候选或执行可行性结果
+ 执行后依据因果产出/销毁事实更新并释放运行时状态
```

## 一、总体判断

从 566 条中文摘要看，暂未发现单篇摘要同时覆盖 P017 的完整组合。

当前近似公开主要分散在以下方向：

1. 长程任务图、技能图、动作契约、参数实例化与安全投影。
2. 跨品牌、跨具身或异构机器人能力抽象与适配。
3. 技能库、示教数据、技能片段、PDDL 或行为蓝图。
4. 当前世界状态、环境状态反馈、执行可行性校验与重规划。
5. 数字孪生、仿真到现实、跨平台迁移。

因此，P017 仍应把区别点集中在“经验资产如何以不变量契约存储，并在当前空间语义、本体能力画像和任务期 RuntimeWorldState 的共同约束下生成适配可行性结果”上，避免写成普通任务规划、技能库调用、跨本体策略学习或轨迹重定向。

## 四、暂不建议抓全文

以下类型摘要较多，但对 P017 独权主链压力较低，可暂作背景噪声处理：

1. 仅涉及路径规划、导航避障、语义地图构建或 3D 场景交互的方案。
2. 仅涉及示教数据采集、轨迹学习、动作重定向、模仿学习、VLA 训练数据生成的方案。
3. 仅涉及机器人硬件结构、末端执行器拆装、逆运动学、轴不变量或动力学建模的方案。
4. 仅涉及多智能体任务分配、云边调度、通用 Agent 工作流或业务流程编排的方案。

特别说明：CN109079784A、CN108942942A、CN108972558A 等出现“不变量”，但其不变量是多轴机器人建模、逆运动学或动力学中的轴不变量，不是 P017 的经验不变量契约，不建议优先抓全文。

## 五、对 P017 撰写的反向提示

1. 独权不要写成“基于当前世界状态进行任务图搜索”。
2. 独权不要写成“技能图规划 + 参数实例化 + 安全投影”。
3. 独权不要写成“跨本体策略学习、动作令牌或轨迹重定向”。
4. 独权不要写成“技能库含前提/后果/成功条件”。
5. 必须守住“经验不变量契约 + 当前空间语义 + 本体能力画像 + 任务期 RuntimeWorldState + 适配可行性结果 + 执行期状态更新与释放”的组合。
