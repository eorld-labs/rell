# P017 中文组合检索词 v2

## 一、检索说明

本清单供中文专利、论文、项目和产业资料检索使用。建议不要只跑单词，而是按组合关系检索。若平台支持高级检索，可将同组词用 AND 关系连接，不同组之间用 OR 或分轮检索。

P017 当前中文主术语建议为：

```text
经验不变量契约
本体能力画像
跨本体适配判断
执行可行性结果
运行时世界状态快照
任务期工作记忆
经验步骤执行可行性判断
```

避免把 P017 主线写成：

```text
本体准入
跨本体准入
主体准入
空间准入
```

## 五、判断提醒

中文检索时，如果只检到“技能前置条件/后置效果”“任务规划/世界状态”“跨本体策略迁移”“示范学习轨迹模型”，不一定破坏 P017。真正需要警惕的是同时出现以下组合的文献：

```text
经验记录不保存具体坐标/关节角/时长
+ 经验以拓扑关系、物理约束、事实终止条件保存
+ 当前空间语义
+ 当前执行体本体能力画像
+ 当前任务期运行时世界状态快照
+ 输出执行可行性结果或绑定候选
```
